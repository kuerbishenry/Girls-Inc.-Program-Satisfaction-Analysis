# Girls-Inc.-Program-Satisfaction-Analysis
Final Project collaboration with Girls Inc Worcester

Girls Inc. is a nonprofit organization committed to empowering young women through its three core pillars: Strong, Smart, and Bold. 
This project was motivated by a need to systematically assess how well Girls Inc. is meeting those objectives over time, and to evaluate
the impact of newer program offerings on participant outcomes. By analyzing longitudinal satisfaction data from the SSBOS surveys (2022
-2025) alongside 2026 program-specific survey results, this project provides an evidence-based picture of where programs are succeeding and
where targeted improvements are needed.
